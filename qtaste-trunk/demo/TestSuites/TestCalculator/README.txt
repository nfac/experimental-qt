Demo: Calculator demo
=====================

Introduction
============

This sample demonstrates how to perform test of applications using Windows GUI. Please notice that THIS DEMO IS ONLY WORKING ON MICROSOFT WINDOWS platform.

It shows an integration of the pyWinAuto Open-Source solution using a client XML/RPC as agent for communication with pyWinAuto.

This demo is provided as example and may not run in your own environment without adaptation.

However, it gives a good idea about how to perform automated tests. Please see the Calculator video" in order to see it in action.

Pre-Requisites
==============
- python 2.5 installed and available in the PATH environment variable (the console should starts if you type python in a command prompt)
  Available on http://www.python.org/download/releases/2.5/
- JDK 1.6


Running The Client
==================

Start the GUI client from the demo directory using the "startUI.cmd' or "startUI.sh" commands

 * Select the "demo_gui" as testbed configuration
 * Select the TestSuites called "TestCalculator"
 * Start the test
