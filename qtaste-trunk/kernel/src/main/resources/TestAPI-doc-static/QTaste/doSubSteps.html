doSubSteps

Function to execute an imported sequence of test steps as sub-steps of the current step.

This may be used to execute a steps sequence of a test script imported with importTestScript, as sub-steps.

 
Function Detail

doSubSteps

void doSubSteps(Sequence stepsSequence [, String selector])
Executes the selected steps of the given steps sequence, all steps being executed if no selector is given, as sub-steps of the current step.

Parameters:
stepsSequence - the steps sequence, which is a list or tuple of (stepId, stepFunction) tuples, stepId being an integer or a string.
selector - (optional) the steps selector, using one of the following formats:
  • "[firstStepId-lastStepId]" to execute steps from firstStepId to lastStepId;
  • "[firstStepId-]" to execute steps from firstStepId to last step;
  • "[-lastStepId]" to execute steps from first step to lastStepId;
  • "[stepId]" to execute only step stepId.
