doSubStep

Function to execute an imported test step as a sub-step of the current step.

This may be used to execute a step of a test script imported with importTestScript, as a sub-step.

 
Function Detail

doSubStep

void doSubStep([Integer_or_String stepId,] Function stepFunction)
Executes the stepFunction function as a sub-step of the current step, using stepId as step id if given, or the step number, starting from 1 for the first step, otherwise.

Parameters:
stepId - (optional) the step id, as an integer or a string.
stepFunction - the step function.
