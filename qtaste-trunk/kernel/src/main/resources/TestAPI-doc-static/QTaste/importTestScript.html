importTestScript

Function to import another test script in order to reuse its steps using the doSubStep and doSubSteps functions.

 
Function Detail

importTestScript

void importTestScript(String testScriptPath)
Imports the test script located in the testScriptPath directory and make its symbols, including step functions and steps sequences, available in a namespace named as the test case name, i.e. the name of the final directory of the test script.

Actually, the code of the test script is evaluated except for the doStep and doSteps functions which are automatically skipped to avoid executing the steps during the import.
The isInTestScriptImport function may also be used to avoid executing some code during a test script import.

Parameters:
testScriptPath - path to the directory of the test script to import, relative to the parent directory of the directory of the current test script.
It is advised to used forward slashes (/) as directory separator instead of backslashes(\) to make path platform independent.
