doSteps

void doSteps(Sequence stepsSequence [, String selector])
Executes the selected steps of the given steps sequence, all steps being executed if no selector is given.

Parameters:
stepsSequence - the steps sequence, which is a list or tuple of (stepId, stepFunction) tuples, stepId being an integer or a string.
selector - (optional) the steps selector, using one of the following formats:
  • "[firstStepId-lastStepId]" to execute steps from firstStepId to lastStepId;
  • "[firstStepId-]" to execute steps from firstStepId to last step;
  • "[-lastStepId]" to execute steps from first step to lastStepId;
  • "[stepId]" to execute only step stepId.
