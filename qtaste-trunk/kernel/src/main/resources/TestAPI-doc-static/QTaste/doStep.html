void doStep([Integer_or_String stepId,] Function stepFunction)
Executes the stepFunction function as a test step, using stepId as step id if given, or the step number, starting from 1 for the first step, otherwise.

Parameters:
stepId - (optional) the step id, as an integer or a string.
stepFunction - the step function.
