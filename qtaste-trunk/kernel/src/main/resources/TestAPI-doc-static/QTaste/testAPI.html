getComponent

Object getComponent([String INSTANCE_ID = instanceId])
Gets an instance of the Test API component named Component.

Parameters:
INSTANCE_ID - (optional) the instance id.
For singleton components, no argument is required; for multiple instances components, a INSTANCE_ID test data is required to define the instance id, it can be defined either as the default instance Id in the testbed configuration, in the CSV file, in the script via the testData variable or by passing it as a named argument.
Returns:
the requested test API component instance, which can be tested for truth value to check if the component is present in the testbed.
