logger

Provides a log4j logger.

The logger level is defined by the log4j.category.TestScript property in the log4j.properties QTaste configuration file.
Valid values are: ALL (log all messages), TRACE, DEBUG, INFO, WARN, ERROR, FATAL, OFF (don't log any message).
The log levels are ordered: TRACE < DEBUG < INFO < WARN < ERROR < FATAL.
A log method actually logs a message only if the log level is higher or equal than the logger level.

Only most useful methods are documented here, see org.apache.log4j.Logger for complete documentation.
